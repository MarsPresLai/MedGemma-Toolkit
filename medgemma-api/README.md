# MedGemma API

The API provides the core functionality and connection to the underlying medical language model.

## Setup

1.  Navigate to the API directory:
    ```bash
    cd medgemma-api
    ```
2.  Install the required Python packages:
    ```bash
    pip install -r requirements.txt
    ```

## Running the API

1.  From the `/medgemma-api` directory, start the server:
    ```bash
    python main.py
    ```
2.  The API will be running on your local machine, typically at `http://127.0.0.1:8000`.
