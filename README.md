# MedGemma: API & CLI

This repository contains the backend API and command-line interface (CLI) for MedGemma, a tool designed to interact with medical language models.

## Project Structure

This project contains two main packages. Please refer to the `README.md` within each directory for specific setup and usage instructions.

-   [`/medgemma-api`](./medgemma-api/): The backend server built with Python.
-   [`/medgemma-cli`](./medgemma-cli/): The command-line client for interacting with the API.
